1. What is the definition of a server port?
A server port is a numbered communication endpoint used by a server to listen for and manage incoming network connections. It allows multiple services to operate on a single IP address.

2. What is server port range?
The server port range is from 0 to 65535.

3. How many bits does the number above represent?
The port number range is represented using 16 bits.

4. State 10 common server ports:
1) HTTP – 80
2) HTTPS – 443
3) FTP – 21
4) SSH – 22
5) SMTP – 25
6) DNS – 53
7) POP3 – 110
8) IMAP – 143
9) MySQL – 3306
10) RDP – 3389
